・How to Use Lilithport to Play Wonderful World Online [English]

Before following these instructions, ensure you've downloaded and extracted Lilithport. You can download it here or here. If your operating system is Windows 10, you are required to have .NET Framework 3.5.



1. Launch Lilithport.exe. You'll be greeted by a server landing window. Press the button highlighted below and click OK.





2. On the subsequent window, press the uppermost ellipsis button and select your WonderfulWorld_ver_XXXX.exe file.



3. In boxes highlighted below, write your profile name and save it with the righmost button beside it, then tick the checkbox to autoarrange the Replay file list.



4. Once you've completed those steps, press the ゲーム ["Game"] tab and complete the following instructions (retain the same format here):



  最大ステージ数   Maximum Number of Stages
Set this to 1. Wonderful World only contains one stage per game.
  ランダムステージ   Random Stages
Set this to 0 if that's not its default.
  ラウンド数   Round Numbers.
Set this to 2 if that's not its default.
  ゲームタイマー   Game Timer.
Set this to 0 if that's not its default.
Do not alter this number; The timer will compromise Wonderful World.
  ラウンド間でHPを持ち越す   Carry Over HP Between Rounds.

Ensure this option is unchecked.


5. Your setup should be complete! To ensure you've correctly setup Lilithport, on the launching page, type the "/game" command and hit enter. The game should successfully boot up.






・ How to Connect to the Lobby:

1. Boot up Lilithport.exe [or type the "/restart" command and hit enter if Lilithport.exe is active] and follow these instructions (you've got the format):



  ニックネーム  Player Name
  コメント  Custom Player Message
  接続先アドレス   The Server IP Address. Only Base64 encoding is supported. The Wonderful World lobby is "yukari.mydns.jp:60512".

2. Once you've entered the lobby IP Address and checked your information, press the 接続 ["Connect"] button. You'll be taken to this Lobby Window:



  ①  
This is the Member List. Here's what the different highlights mean:
 No Highlight - The user is idling.
 Yellow Highlight - The user is in Rest Mode. While in this mode, they cannot be fought. To enter this mode yourself, type "/rest" in the chat.
 Blue Highlight - The user is actively searching for a match.
 Red Highlight - The user is currently in a match.
 Green Highlight - The user is currently spectating a match.
Green nickname - The Lobby Owner
Blue nickname - Host
Black nickname - User/Client

  ②   		This is the chatlog. All messages will appear here.

  ③   		Goes without saying, this is where you type messages.

  ④   		This is the Game Profile. You can manage game settings through here (do be cautious that the only tab may be "Lilithport". That should mean everything's correct.)

3. To challenge a player or spectate a match, right click their name. A handful of options will show up:



  ① 対戦する ["Challenge"]   Challenge the player to a battle.
  ② 観戦する ["Watch the Match"]   Spectate an ongoing match. Only available if user has a red highlight.
  ③対戦募集中する ["Looking for a Battle"]   Click this to highlight your name blue. To turn it off, click the button again.
  ④ PING   PThis button will calculate the ping between you and the user. The final calculation will appear in the chatbox. This will give you an idea of the necessary delay.
  ⑤名前をコピー ["Copy Name"]   Copies the user's name to the clipboard.

Miscellaneous Information:

-Lilithport automatically assigns you to your Player 1 controls, regardless of your side. Ensure your preferred control scheme has been saved to the game.
